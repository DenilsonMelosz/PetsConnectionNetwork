# 🐾 Pets Connection Network (PCN)

> Rede social voltada à adoção responsável e à localização de animais desaparecidos.

---

## 🌟 Visão Geral

O **Pets Connection Network (PCN)** é uma aplicação web desenvolvida para conectar **ONGs, tutores e adotantes**.  
A plataforma tem como objetivo principal **facilitar a adoção de pets** e **auxiliar na busca por animais perdidos**, criando um ambiente seguro, empático e digitalmente acessível.

---

## 🖥️ Estrutura e Páginas

O sistema conta com quatro áreas principais:

- **Tela de Login e Cadastro** — acesso seguro, com autenticação individual.
- **Perfil de Usuário** — três tipos de perfis: **ONG**, **Usuário Comum** e **Administrador**.
- **Feed de Postagens** — espaço para divulgar pets disponíveis, desaparecidos ou encontrados.
- **Tela de Adoção e Pesquisa** — permite localizar animais por filtros específicos (raça, cidade, status, etc.).
